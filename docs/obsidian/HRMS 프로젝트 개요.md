# HRMS 프로젝트 개요

## 프로젝트 정보

| 항목 | 내용 |
|------|------|
| 프로젝트명 | PANASIA HRMS (인사관리시스템) |

## 인증 체계

- **AuthGuard**: 미인증 사용자 → `/login` 자동 리다이렉트
- **역할 기반 접근제어 (RBAC)**
  - `admin` - 시스템관리자 (전체 접근)
  - `hr_manager` - 인사담당자
  - `dept_manager` - 부서관리자
  - `employee` - 일반사원
- 메뉴별 역할 권한 설정 가능 ([[설정]] 참조)
