# 감사로그

> 경로: `/audit-log` | 파일: `src/app/audit-log/page.tsx`

## 개요

시스템 내 모든 사용자 활동을 기록하고, 컴플라이언스 감사를 위한 조회/필터링을 제공한다.

## 주요 기능

- **통계**: 전체 로그, 금일 페이지뷰, 금일 액션, 활성 사용자
- **필터**: 사용자명, 액션 유형, 날짜 범위
- **로그 테이블**: 시간, 사용자, 역할, 액션, 대상
- **전체 삭제**: 확인 후 로그 초기화
- **자동 추적**: `PageViewTracker` 컴포넌트

## 액션 유형

| 액션 | 코드 | 아이콘 |
|------|------|--------|
| 페이지 조회 | `page_view` | Eye |
| 생성 | `create` | Plus |
| 수정 | `update` | Edit |
| 삭제 | `delete` | Trash |
| 로그인 | `login` | LogIn |
| 로그아웃 | `logout` | LogOut |
| 내보내기 | `export` | Download |
| 승인 | `approve` | Check |
| 반려 | `reject` | X |
