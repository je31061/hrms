# 교육관리

> 경로: `/training`, `/training/[id]` | 파일: `src/app/training/`

## 개요

사내/외 교육 프로그램을 등록하고, 수강 신청 및 이수 관리를 수행한다.

## 주요 기능

- **통계**: 전체, 진행중, 예정, 완료
- **프로그램 카드**: 상태, 강사, 장소, 수강 인원 표시
- **프로그램 상세** (`/training/[id]`): 수강생 목록, 이수 현황
- 프로그램 등록 기능

## 교육 상태

| 상태 | 코드 | 색상 |
|------|------|------|
| 예정 | `planned` | 황색 |
| 진행중 | `in_progress` | 파랑 |
| 완료 | `completed` | 초록 |
| 취소 | `cancelled` | 빨강 |

## 데이터 모델

```typescript
Training {
  id: string
  title: string
  category: string
  instructor: string
  location: string
  start_date: string
  end_date: string
  max_participants: number
  status: TrainingStatus
}

TrainingEnrollment {
  id: string
  training_id: string
  employee_id: string
  status: 'enrolled' | 'completed' | 'cancelled'
  completion_date?: string
  score?: number
}
```
