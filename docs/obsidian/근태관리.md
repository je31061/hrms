# 근태관리

> 경로: `/attendance`, `/attendance/monthly` | 파일: `src/app/attendance/`

## 개요

임직원의 출퇴근 기록을 관리하고 일별/월별 근태 현황을 파악한다.

## 주요 기능

### 일일 근태 (`/attendance`)
- **출퇴근 버튼**: 원클릭 출근/퇴근 기록
- **금일 통계**: 출근유형별 집계
  - 출근, 출장, 외근, 재택, 지각, 결근, 휴가
- **금일 근태 테이블**: 전 직원 출퇴근 현황
- **수동 등록**: 근태 등록 다이얼로그

### 월별 현황 (`/attendance/monthly`)
- 월 단위 근태 캘린더 뷰
- 개인별 월간 근태 요약

## 근태 상태 분류

| 상태 | 코드 | 설명 |
|------|------|------|
| 정상 | `normal` | 정시 출근 |
| 지각 | `late` | 출근 시간 초과 |
| 조퇴 | `early_leave` | 퇴근 시간 미달 |
| 결근 | `absent` | 출근 기록 없음 |
| 휴일 | `holiday` | 공휴일/주말 |
| 휴가 | `leave` | 승인된 휴가 |

## 근무 유형

| 유형 | 코드 |
|------|------|
| 사무실 | `office` |
| 출장 | `business_trip` |
| 외근 | `field_work` |
| 재택 | `remote` |

## 데이터 모델

```typescript
Attendance {
  id: string
  employee_id: string
  date: string           // YYYY-MM-DD
  clock_in?: string      // HH:mm
  clock_out?: string     // HH:mm
  work_hours: number
  overtime_hours: number
  status: AttendanceStatus
  attendance_type: AttendanceType
  location?: string
  purpose?: string
  note?: string
}
```
