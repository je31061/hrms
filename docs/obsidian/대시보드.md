# 대시보드

> 경로: `/` | 파일: `src/app/page.tsx`

## 개요

HRMS의 메인 화면으로, 핵심 인사 지표와 최근 이벤트를 실시간으로 표시한다.

## 주요 기능

### 통계 카드 (4개)

| 카드 | 내용 | 아이콘 |
|------|------|--------|
| 전체 임직원 | 정규직/비정규 구분 | Users |
| 금일 출근 | 출근율 % | Clock |
| 휴가 중 | 승인된 휴가 기준 | CalendarDays |
| 금년 입퇴사 | 이직률 % | UserRoundPlus |

### 차트

- **부서별 인원현황**: Level 2 부서 기준 막대 차트
- **입퇴사 추이**: 최근 12개월 입사/퇴사 라인 차트

### 최근 이벤트

- 발령 이력 (입사/승진/전보/퇴사)
- 승인된 휴가
- 최신 5건 표시

## 컴포넌트 구조

```
DashboardPage
├── Breadcrumb
├── StatsCard (x4)
├── HeadcountChart (부서별 인원)
├── HireResignChart (입퇴사 추이)
└── RecentEvents (최근 이벤트)
```
