# 로그인

> 경로: `/login` | 파일: `src/app/login/page.tsx`

## 개요

인증 및 회사 정보를 표시하는 랜딩 페이지. 글래스모피즘 UI와 인터랙티브 애니메이션을 적용했다.

## 인증 기능

### 데모 모드 (Supabase 미설정 시)
- **빠른 로그인**: 역할별 원클릭 로그인
- **이메일 로그인**: 데모 계정 이메일/비밀번호
- 비밀번호: `demo1234`

### Supabase 모드
- 이메일/비밀번호 인증
- Supabase Auth 연동

## 데모 계정

| 역할 | 이름 | 설명 |
|------|------|------|
| admin | 시스템관리자 | 전체 접근 권한 |
| hr_manager | 인사담당자 | 인사 관련 메뉴 |
| dept_manager | 부서관리자 | 부서 관련 메뉴 |
| employee | 일반사원 | 기본 메뉴만 |

## 랜딩 페이지 섹션

스크롤 시 아래 섹션이 순차적으로 노출된다:

1. **Hero (로그인 폼)** - 로그인 + Globe 애니메이션
2. **Live Data** - 날씨, 환율, 시장지표
3. **News** - 주요 뉴스 헤드라인
4. **Industry** - 해운/조선/환경 동향
5. **Company & Features** - 파나시아 소개 + HRMS 기능
6. **World Clock** - 글로벌 거점 현지 시각
7. **Footer** - 회사 정보

## 비주얼 효과

- `ParticleCanvas` - 파티클 네트워크 애니메이션
- `FloatingPlanes` - 부유하는 비행기 SVG
- `Globe` - 회전하는 지구본 캔버스
- `Glass` - 글래스모피즘 카드 컴포넌트
- `useReveal()` - 스크롤 시 fade-in 효과
