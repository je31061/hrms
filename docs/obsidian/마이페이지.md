# 마이페이지

> 경로: `/my` | 파일: `src/app/my/page.tsx`

## 개요

로그인한 직원 본인의 인사정보를 통합 조회하는 개인 포털이다.

## 탭 구성

### 1. 기본정보
- 개인 상세정보 그리드
- 증명서 발급 링크 (재직증명서, 경력증명서)

### 2. 인사발령
- 전체 발령 이력
- 변경 전/후 직급/부서 표시

### 3. 연차/휴가
- 유형별 잔여일수 프로그레스 바
- 휴가 신청 이력
- 새 휴가 신청 버튼

### 4. 급여
- 현재 기본급 표시
- 급여 이력 (상태별)
- 급여명세서 조회

### 5. 근태
- 최근 근무일수 요약 (정상/지각/기타)
- 상세 근태 기록 (최근 10건)
- 근무시간 계산

### 6. 교육/평가
- 교육 이수 이력 (점수 포함)
- 평가 결과 (등급, 평가자 코멘트)

## 컴포넌트 구조

```
MyPage
├── ProfileCard (프로필 + 퀵 스탯)
└── Tabs
    ├── 기본정보 (InfoGrid)
    ├── 인사발령 (AppointmentTable)
    ├── 연차/휴가 (LeaveBalanceCards + LeaveHistory)
    ├── 급여 (PayrollTable)
    ├── 근태 (AttendanceSummary + AttendanceTable)
    └── 교육/평가 (TrainingHistory + EvalResults)
```
