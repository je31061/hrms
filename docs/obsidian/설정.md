# 설정

> 경로: `/settings` | 파일: `src/app/settings/page.tsx`

## 개요

시스템 전반의 기준정보와 정책을 설정하는 관리자 모듈이다. 총 22개 설정 탭으로 구성된다.

## 설정 항목

### 기본 설정
| # | 탭 | 설명 |
|---|-----|------|
| 1 | 회사정보 | 회사명, 사업자번호, 로고 등 |
| 2 | 부서관리 | 부서 계층 구조 CRUD |
| 3 | 직급관리 | 직급 체계 설정 |
| 4 | 직책관리 | 직책 체계 설정 |
| 5 | 직무관리 | 직무 분류 코드 |
| 6 | 호봉관리 | 직급별 호봉 테이블 |

### 근무/급여 설정

|---|-----|------|

## 공통 패턴

모든 설정 항목에 적용되는 공통 기능:
- **사용 유무** (`is_active`): 활성/비활성 토글
- **적용기간** (`effective_from` / `effective_to`): From/To 날짜
- **변경이력**: 누가, 언제, 무엇을 변경했는지 자동 추적
