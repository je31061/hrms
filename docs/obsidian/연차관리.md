# 연차관리

> 경로: `/leave`, `/leave/admin` | 파일: `src/app/leave/`

## 개요

임직원의 연차/휴가 잔여일수를 관리하고, 휴가 신청/승인 프로세스를 운영한다.

## 주요 기능

### 휴가 현황 (`/leave`)
- **잔여일수 카드**: 유형별 시각적 프로그레스 바
- **휴가 신청**: 모달 폼으로 휴가 요청
- **신청 이력**: 상태별 테이블 (대기/승인/반려/취소)
- **취소 기능**: 대기/승인 상태 건 취소 가능

### 관리자 설정 (`/leave/admin`)
- 휴가 유형별 일수 설정
- 잔여일수 일괄 조정

## 휴가 유형

| 유형 | 코드 | 유급여부 | 최대일수 |
|------|------|----------|----------|
| 연차 | `annual` | 유급 | 근속연수별 |
| 병가 | `sick` | 유급 | 60일 |
| 경조사 | `condolence` | 유급 | 사유별 |
| 출산휴가 | `maternity` | 유급 | 90일 |
| 배우자출산 | `paternity` | 유급 | 10일 |

## 휴가 신청 상태

| 상태 | 코드 | 다음 상태 |
|------|------|-----------|
| 대기 | `pending` | approved / rejected |
| 승인 | `approved` | cancelled |
| 반려 | `rejected` | - |
| 취소 | `cancelled` | - |
