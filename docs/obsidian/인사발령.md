# 인사발령

> 경로: `/appointments`, `/appointments/new` | 파일: `src/app/appointments/`

## 개요

임직원의 인사발령(승진, 전보, 입사, 퇴사 등)을 등록하고 이력을 관리한다.

## 주요 기능

- 발령 통계: 전체, 승진, 전보 건수
- 발령 테이블: 유형별 뱃지, 변경 전/후 표시
- 발령 등록 (`/appointments/new`)

## 발령 유형

| 유형 | 코드 | 색상 |
|------|------|------|
| 승진 | `promotion` | 초록 |
| 전보 | `transfer` | 보라 |
| 직책변경 | `title_change` | 황색 |
| 입사 | `hire` | 파랑 |
| 퇴사 | `resignation` | 빨강 |

## 데이터 모델

```typescript
Appointment {
  id: string
  employee_id: string
  type: 'promotion' | 'transfer' | 'title_change' | 'hire' | 'resignation'
  effective_date: string
  prev_department_id?: string
  prev_position_rank_id?: string
  new_department_id?: string
  new_position_rank_id?: string
  reason?: string
  prev_salary?: number
  new_salary?: number
}
```
