# 인사정보 관리

> 경로: `/employees`, `/employees/[id]`, `/employees/new` | 파일: `src/app/employees/`

## 개요

전체 임직원의 인사정보를 등록/조회/수정하는 핵심 모듈이다.

## 주요 기능

### 직원 목록 (`/employees`)
- 재직/퇴직 필터링
- 통계 카드: 전체, 재직, 휴직, 퇴직
- 정렬/필터 가능한 직원 테이블
- 부서, 직급, 직책 표시

### 직원 상세 (`/employees/[id]`)
- 기본정보 상세 조회/수정
- 경력사항, 학력, 자격증, 가족관계 탭

### 신규 등록 (`/employees/new`)
- 직원 정보 입력 폼
- 부서/직급/직책 선택

## 컴포넌트 구조

```
EmployeesPage
├── Breadcrumb
├── StatsCard (x4)
├── 필터 (재직상태, 고용형태)
└── EmployeeTable
    ├── 검색
    ├── 정렬
    └── 페이지네이션
```

## 데이터 모델

```typescript
Employee {
  id: string
  employee_number: string    // 사번
  name: string               // 이름
  name_en?: string           // 영문명
  email: string
  phone: string
  birth_date: string
  gender: 'male' | 'female'
  address: string
  department_id: string      // → Department
  position_rank_id: string   // → PositionRank (직급)
  position_title_id: string  // → PositionTitle (직책)
  employment_type: 'regular' | 'contract' | 'part_time' | 'intern'
  hire_date: string
  resignation_date?: string
  status: 'active' | 'on_leave' | 'resigned'
  base_salary: number
  profile_image_url?: string
  bank_name?: string
  bank_account?: string
}
```
