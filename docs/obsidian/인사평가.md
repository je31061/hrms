# 인사평가

> 경로: `/evaluation`, `/evaluation/[id]` | 파일: `src/app/evaluation/`

## 개요

평가 기간을 설정하고, 다면 평가(자기/상사/동료)를 진행하며, 등급을 관리한다.

## 주요 기능

- **평가 기간 목록**: 상태별 카드 표시
- **완료율 프로그레스 바**
- **평가 기간 정보**: 기간, 평가자 수
- **평가 상세** (`/evaluation/[id]`): 개인별 평가 결과
- 새 평가 기간 생성

## 평가 상태

| 상태 | 코드 | 색상 |
|------|------|------|
| 작성중 | `draft` | 회색 |
| 진행중 | `in_progress` | 파랑 |
| 완료 | `completed` | 초록 |

## 평가 구성

### 평가 유형
- 자기 평가 (Self)
- 상사 평가 (Superior)
- 동료 평가 (Peer)

### 평가 등급
- S (최우수), A (우수), B (양호), C (보통), D (미흡)
