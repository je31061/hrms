# 조직도

> 경로: `/organization`, `/organization/departments` | 파일: `src/app/organization/`

## 개요

회사 조직 구조를 시각적으로 표시하고, 시뮬레이션을 통해 조직 개편을 미리 검토할 수 있다.

## 주요 기능

### 조직도 시각화
- **트리 뷰**: 좌측 계층형 부서 목록
- **차트 뷰**: 우측 비주얼 조직도 캔버스
- 부서 카드 클릭 시 소속 직원 표시
- 직원 프로필 사진 (DiceBear 아바타 자동 생성)

### 시뮬레이션 모드
- 부서/직원 드래그 앤 드롭
- Undo / Reset 기능
- 변경 이력 추적
- 조직 개편 미리보기

### 부서 관리 (`/organization/departments`)
- 부서 추가/수정/삭제
- 계층 구조 관리

## 컴포넌트 구조

```
OrganizationPage
├── OrgTree (좌측 트리)
├── OrgChartCanvas (우측 차트)
│   ├── DepartmentCard
│   └── DragOverlay
├── SimulationToolbar
└── EmployeeList (부서별)
```

## 핵심 유틸리티

| 함수 | 기능 |
|------|------|
| `buildDeptTree()` | 평면 부서 → 트리 변환 |
| `buildEmployeesByDept()` | 부서별 직원 그룹핑 |
| `reparentDepartment()` | 부서 이동 |
| `moveEmployee()` | 직원 부서 이동 |
| `recomputeCounts()` | 인원 수 재계산 |

## 데이터 모델

```typescript
Department {
  id: string
  name: string
  code: string
  parent_id: string | null
  level: number        // 1: 본부, 2: 부서, 3: 팀
  sort_order: number
  is_active: boolean
  effective_from?: string
  effective_to?: string
  children?: Department[]
}
```
