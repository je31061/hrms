# 채용관리

> 경로: `/recruitment`, `/recruitment/[id]`, `/recruitment/new` | 파일: `src/app/recruitment/`

## 개요

채용 공고 등록부터 지원자 추적까지 전체 채용 프로세스를 관리한다.

## 주요 기능

- **통계**: 전체 공고, 진행중, 지원자 수, 채용 목표
- **공고 카드**: 상태, 마감일, 지원자 수 표시
- **공고 상세** (`/recruitment/[id]`): 지원자 리스트, 전형 단계 관리
- **공고 등록** (`/recruitment/new`): 새 채용 공고 작성

## 공고 상태

| 상태 | 코드 | 색상 |
|------|------|------|
| 진행중 | `open` | 초록 |
| 작성중 | `draft` | 황색 |
| 마감 | `closed` | 회색 |
| 취소 | `cancelled` | 빨강 |

## 지원자 전형 단계

서류전형 → 1차 면접 → 2차 면접 → 최종합격
