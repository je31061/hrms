# i18n (한/영 옵션) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** HRMS의 크롬 영역(사이드바, 헤더, 알림벨)에 한글/영문 토글 기능을 추가하고, 향후 페이지 본문 번역 확장의 기반이 되는 i18n 인프라를 구축한다.

**Architecture:** 외부 i18n 라이브러리를 도입하지 않고 React hook(`useT`) + 평탄화된 dictionary(`ko.ts`, `en.ts`) 조합으로 자체 구현한다. Locale은 기존 `settings-store.display`에 추가하여 Zustand persist를 그대로 활용하고, 헤더 우측 드롭다운에서 전환한다. 영문 사전 누락 시 한글로 폴백하고, dev에서 콘솔 경고를 띄운다.

**Tech Stack:** Next.js 16 (App Router) / React 19 / TypeScript / Zustand 5 (+ persist) / shadcn-ui (Radix) / lucide-react / tailwindcss 4.

**테스트 환경:** 프로젝트에 자동 테스트 프레임워크가 없으므로, 각 Task는 다음 두 가지로 검증한다:
- `npm run lint`와 `npm run build`로 타입/린트 오류 확인 (자동)
- `npm run dev` 후 브라우저에서 명시된 시나리오 확인 (수동)

### Task 4: useT() hook 구현

**Files:**
- Create: `src/lib/i18n/use-translation.ts`

- [ ] **Step 1: hook 구현 — 폴백/인터폴레이션 포함**

```typescript
// src/lib/i18n/use-translation.ts
'use client';

import { useSettingsStore } from '@/lib/stores/settings-store';
import { ko } from './dictionaries/ko';
import { en } from './dictionaries/en';
import type { Locale, TranslationKey, Dictionary } from './types';

const DICTIONARIES: Record<Locale, Dictionary> = { ko, en };

function interpolate(template: string, vars?: Record<string, string | number>): string {
  if (!vars) return template;
  let out = template;
  for (const [k, v] of Object.entries(vars)) {
    out = out.replace(new RegExp(`\\{\\{${k}\\}\\}`, 'g'), String(v));
  }
  return out;
}

export function useT() {
  const locale = useSettingsStore((s) => s.display.locale);
  const t = (key: TranslationKey, vars?: Record<string, string | number>): string => {
    const value = DICTIONARIES[locale][key] ?? DICTIONARIES.ko[key];
    if (value === undefined) {
      if (process.env.NODE_ENV === 'development') {
        // eslint-disable-next-line no-console
        console.warn(`[i18n] missing translation key: ${key}`);
      }
      return key;
    }
    return interpolate(value, vars);
  };
  return { t, locale };
}
```

> 주의: 이 단계에서 `s.display.locale` 참조는 settings-store 변경(Task 5) 전이라 TS 에러가 난다. Task 5에서 store에 `locale` 필드를 추가하면 해결된다. **이 Task에서 lint/tsc 검증은 생략**하고, Task 5 완료 후 한 번에 검증한다.

```typescript
'use client';

import { useEffect } from 'react';
import { useTheme } from 'next-themes';
import { useSettingsStore } from '@/lib/stores/settings-store';

const FONT_SIZE_CLASS: Record<string, string> = {
  small: 'font-sm',
  medium: 'font-md',
  large: 'font-lg',
};

const DENSITY_CLASS: Record<string, string> = {
  compact: 'density-compact',
  comfortable: 'density-comfortable',
  spacious: 'density-spacious',
};

export function DisplaySettingsApplier() {
  const display = useSettingsStore((s) => s.display);
  const { setTheme } = useTheme();

  useEffect(() => {
    setTheme(display.theme);
  }, [display.theme, setTheme]);

  useEffect(() => {
    const html = document.documentElement;
    Object.values(FONT_SIZE_CLASS).forEach((cls) => html.classList.remove(cls));
    html.classList.add(FONT_SIZE_CLASS[display.font_size] ?? 'font-md');

    Object.values(DENSITY_CLASS).forEach((cls) => html.classList.remove(cls));
    html.classList.add(DENSITY_CLASS[display.content_density] ?? 'density-comfortable');
  }, [display.font_size, display.content_density]);

  useEffect(() => {
    document.documentElement.lang = display.locale;
  }, [display.locale]);

  return null;
}
```

기존 `timeAgo`는 외부 함수이므로 hook을 쓸 수 없다. `NotificationBell` 컴포넌트 안에서 사용되도록 클로저로 만든다. 파일 상단의 `function timeAgo(iso: string): string` 정의를 **삭제**하고, `NotificationBell` 컴포넌트 본문 내부에 `const t`를 받은 뒤 다음 함수를 정의:

```typescript
const { t, locale } = useT();

const timeAgo = (iso: string): string => {
  const now = new Date();
  const then = new Date(iso);
  const diffMs = now.getTime() - then.getTime();
  const diffMin = Math.floor(diffMs / (1000 * 60));
  if (diffMin < 1) return t('notification.timeAgo.justNow');
  if (diffMin < 60) return t('notification.timeAgo.minutes', { n: diffMin });
  const diffHour = Math.floor(diffMin / 60);
  if (diffHour < 24) return t('notification.timeAgo.hours', { n: diffHour });
  const diffDay = Math.floor(diffHour / 24);
  if (diffDay < 7) return t('notification.timeAgo.days', { n: diffDay });
  return then.toLocaleDateString(locale === 'ko' ? 'ko-KR' : 'en-US');
};
```
